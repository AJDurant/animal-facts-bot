# animal-facts-bot

A Reddit bot that searches for comments on reddit that contain the name of the animal then replies to the comment with a fact about that animal.

### Current supported animals:
* alligator
* badger
* camel
* dolphin
* flamingo
* frog
* giraffe
* hippo
* horse
* jellyfish
* koala
* monkey
* octopus
* owl
* penguin
* pig
* scorpion
* sloth
* snake
* tiger
* turtle
* wolf
* whale
* zebra
